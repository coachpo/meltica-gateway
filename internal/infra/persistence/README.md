# Persistence Layer Overview

Packages under this directory contain Meltica's durable storage abstractions.

- `store.go` exposes the root `Store` wiring that runtime components consume.
- `postgres/` holds PostgreSQL-specific implementations generated by `sqlc` alongside handwritten helpers.

To regenerate query bindings:

```bash
sqlc generate
```

The command uses `sqlc.yaml` at the repository root and emits code into `internal/infra/persistence/postgres/sqlc/`.
